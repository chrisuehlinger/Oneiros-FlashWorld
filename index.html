FlashWorld Spark Demo ! Front-end real-time rendering in Spark uses compressed and pruned Gaussians. Visual quality in this demo may be lower than offline/back-end rendering.

Paper Code Project Page

1. Configure

Set FOV and Resolution and Click "Fix Configurations"

Important: You need to specify your Hugging Face Access Token with READ permission to use the online free ZeroGPU service.

2. Set Camera Trajectory

Manual: Navigate with mouse and keyboard, press Space to record

Template: Select template type and click "Generate Trajectory"

JSON: Load trajectory from JSON file

3. Add Prompts

Upload image or enter text description

4. Generate

Click "Generate!" to create your scene

Controls

Mouse/QE: Rotate view

WASD/RF: Move

Space: Record camera
